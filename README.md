Git Assignment

Overview
This repository contains practical Git and GitHub operations performed as part of the assignment.

The assignment includes:
- Git initialization
- Local and remote repository operations
- Staging and unstaging files
- Branch creation and merging
- Pull requests
- Merge conflict resolution
- Reset operations
- Revert operations
- Stash operations
- Fork simulation
- Branch comparison and cleanup
